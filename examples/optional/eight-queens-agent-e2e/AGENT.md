# Agent Instructions

- Read the repository itself before making changes.
- Implement the fix in `src/eight_queens.py`.
- Run `python3 -m pytest -q` before you commit.
- Before you push, remove generated Python artefacts such as `__pycache__/` and `*.pyc` if they were created.
- Verify the worktree is clean with `git status --short` before you push.
- Commit only on your `agent/<tool>` branch and push only to that branch.
- Never push to `main`.
